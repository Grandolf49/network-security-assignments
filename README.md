# Network Security Assignments

A repository that contains Java programs of Network Security assignments.

## Assignments
1. [Ceaser Cipher](https://en.wikipedia.org/wiki/Caesar_cipher)
2. [PlayFair Cipher](https://en.wikipedia.org/wiki/Playfair_cipher)
3. [Vigenere Cipher](https://en.wikipedia.org/wiki/Vigen%C3%A8re_cipher)

## How to use
1. Clone the project
2. Import the project in Intellij IDEA
3. Run the main method of any class
